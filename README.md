# facephi-challenge-2026
Sistema avanzado de detección y prevención de fraude en documentos de identidad desarrollado para el Facephi Challenge 2026. Integra validación MRZ, análisis de errores reales de OCR y detección de incoherencias internas para identificar manipulaciones sutiles no evidentes.

## Instrucciones de ejecución

Para ejecutar correctamente el notebook es necesario descargar el repositorio completo, incluyendo la carpeta `data` que contiene las imágenes de prueba.

Pasos:

1. Descargar el repositorio completo (Code → Download ZIP).
2. Mantener la estructura original de carpetas.
3. Abrir el archivo `Vidarte_Albarracin_FacephiChallenge2026.ipynb` en Jupyter o Google Colab.
4. Ejecutar todas las celdas en orden.

El notebook se entrega completamente ejecutado y con resultados reproducibles.
